Developer: Hein Htoo Naing

I created this web-based recreation of Five Nights at Freddy's purely for fun and to bring back the enjoyable, terrifying nostalgia of the original 2014 classic. The original game proved that you don't need complex 3D graphics to create a truly tense atmosphere, and I wanted to capture that exact feeling right inside a web browser.

This project was built from the ground up to challenge my coding skills, using nothing but raw HTML5, CSS3, and Vanilla JavaScript - no external game engines, libraries, or frameworks were used. Building this taught me a lot about advanced DOM manipulation, managing complex game states, handling timing events, and structuring a multi-page web project from scratch.

Key Game Mechanics:

  • Functional security camera system with TV static effects.
  • Resource management (power drain from doors, lights, and cameras).
  • Frame-by-frame jumpscare animations using CSS/JS logic.
  • Screen transitions and persistent data tracking.
